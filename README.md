# Sunartha Warehouse

```
git clone git@github.com:sustiono/sunartha-warehouse.git
cd sunartha-warehouse
yarn
yarn start
```

Then open http://localhost:3000 in a browser

Live Demo: https://sunartha-warehouse.vercel.app
